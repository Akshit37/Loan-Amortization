# **Loan Amortization Model**

This repository contains an Excel-based loan amortization model designed to calculate repayment schedules, interest costs, and outstanding balances over the loan tenure. The model is useful for personal finance planning, loan comparison, and credit analysis. The model incorporates the following:

- Generates a complete amortization schedule with principal and interest breakdown

- Supports customizable inputs such as loan amount, tenure, and interest rate

- Calculates EMI, total interest paid, and outstanding balance over time

- Provides clear visual summaries to compare different loan scenarios
